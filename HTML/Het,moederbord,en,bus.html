Het moederbod en de Bus Het moederbord is en hele grote hardware interface. Op het moederbord zitten eigenlijk alle onderdelen van een computer verbonden. Het moederbord is er namelijk op genmaakt om al deze onderdelen met elkaar te laten samenwerken dus als een onderdeel niet op het moederbord zit aangesloten werkt die niet. De Bus Dit is eigenlijk het transport middel van data binnen een computer. Een bus zorgt er namelijk voor dat bijvoorbeeld de CPU en de RAM data met elkaar kunnen uitwisselen. Er zijn 3 verschillende soorten delen van een Bus de Adres bus deze zorgt ervoor dat er verschillende geheugencellen uit een RAM kunnen worden opgeroepen. De Control Bus deze bestaat uit 2 delen de enable wire die ervoor zorgt dat er vanuit het RAM data naar de CPU gestuurd kan worden. En je hebt de setwire die ervoor zorgt dat bestaande geheugencellen in het permanente geheugen overgeschreven kan worden. De Databus deze zorgt ervoor dat data uit de CPU naar het RAM gesstuurd wordt
