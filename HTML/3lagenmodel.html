Een computer is bestaat simpel gezegd uit 3 lagen die met elkaar samen werken om de computer te laten werken. Deze 3 lagen zijn: Laag 1 de fysieke laag: De fysieke laag zijn de onderdelen van je computer die je kan aanraken zoals bijvooorbeeld de camera of de CPU. Al deze onderdelen zijn vaak verbonden op een printplaat via hardware interfaces deze zorgen ervoor dat er via elektronische verbindingen gecommuniceerd kan worden tussen de verschillende lagen. Laag 2 de logische laag: Dit is de software van je computer of mobiel. Deze software zorgt ervoor dat je app kan runnen en laten werken. Voorbeelden van dit soort software zijn windows of IOS. Laag 3 de toepassingslaag: De toepassingslaag bestaat vooral uit applicaties en app op je computer zoals facebook en instagram. Een computer kan alleen werken met dezze 3 lagen want een app kan niet direct met de fysieke laag communiceren of andersom.
