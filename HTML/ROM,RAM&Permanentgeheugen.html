Er zijn 3 soorten geheugen het ROM RAM en het permanente geheugen. De RAM Dit is het Random Acces Memory oftewel het werk geheugen van de computer. In dit geheugen wordt data en instructies tijdelijk opgeslagen. De RAM kan alleen zijn geheugen bewaren zolang die stroomstootjes blijft ontvangen zodra de stroom wegvalt verliezen se geheugencellen hun lading. De ROM Dit is het Read Only Memory en wat het grote verschil is met RAM is dat de geheugencellen van het ROM wel hun geheugen bewaren als de stroom wegvalt. Daarom staan op de ROM altijd de instellingen hoe een computer op te starten. Het permanente geheugen We hebben 2 verschillende permanente geheugen het elektronisch geheugen en het magnetisch geheugen. Het flash geheugen is de vorm van elektronisch geheugen dit heet zo omdat dit geheugen in een flits gewist en weer beschreven kan worden. De harde schijf is een vorm van magnetisch geheugen. Deze bestaat uit een luchtdichte verpakking met daarin een aantal schijven van aluminium of glas. Deze zijn bedekt met een laag magnetisch materiaal en draaien met een grote snelheid rond. Elke schijf heeft een lees-en schrijfkop. De schrijfkop kan data in de laag van een plaat aanbrengen in de vorm van een serie enen en nullen. De opslagcapaciteit wordt bepaald door het aantal schijven.
